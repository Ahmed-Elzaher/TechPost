## 🎯 الميزات الرئيسية

### 1️⃣ تحميل الأخبار

عند فتح التطبيق:
1. يعرض شاشة تحميل مع Spinner
2. بعد 2 ثانية يتم تحميل الأخبار
3. تظهر قائمة الأخبار

### 2️⃣ سحب للتحديث (Pull-to-Refresh)

```
المستخدم يسحب الشاشة للأسفل
        ↓
ظهور فاصل التحديث
        ↓
تحديث البيانات
        ↓
اختفاء الفاصل
```

- اسحب الشاشة للأسفل لتحديث الأخبار
- يظهر مؤشر تحميل أخضر
- يتم تحميل أخبار جديدة

### 3️⃣ الضغط على الخبر

```
الضغط على كارت الخبر
        ↓
ظهور SnackBar
        ↓
عرض عنوان الخبر
        ↓
اختفاء بعد 3 ثواني
```

- يظهر رسالة بعنوان الخبر
- تحتوي على حدود خضراء
- تختفي تلقائياً

### 4️⃣ حالات الخطأ

عند حدوث خطأ:
- عرض رسالة خطأ واضحة
- زر "إعادة محاولة" أخضر
- الضغط يحاول تحميل البيانات مجدداً

## 🔄 دورة حياة التطبيق

### البداية

```
1. قائمة التحميل
   └─ جاري تحميل الأخبار...
   
2. بعد التحميل
   └─ قائمة الأخبار
   
3. في حالة الخطأ
   └─ رسالة خطأ + زر إعادة محاولة
```

### الحالات الثلاث

```
┌──────────────────────────────────┐
│         Loading State             │
│   Spinner + "جاري التحميل..."    │
└──────────────────────────────────┘
              ↓
       ┌──────────────────────────────────┐
       │         Loaded State             │
       │    قائمة الأخبار مع الصور       │
       └──────────────────────────────────┘
              ↓
       ┌──────────────────────────────────┐
       │         Error State              │
       │   رسالة خطأ + زر إعادة محاولة   │
       └──────────────────────────────────┘
```

## 🆘 حل المشاكل

### المشكلة: الأخبار لا تحمل

**الحل:**
1. انتظر قليلاً (يستغرق 2 ثانية)
2. حاول سحب الشاشة للتحديث
3. اضغط زر "إعادة محاولة"

### المشكلة: الصور لا تظهر

**الحل:**
1. تأكد من اتصالك بالإنترنت
2. أعد تحميل التطبيق
3. قد تكون عملية محاكاة

### المشكلة: التطبيق بطيء

**الحل:**
1. أغلق التطبيقات الأخرى
2. امسح ذاكرة التطبيق
3. أعد تشغيل الهاتف
